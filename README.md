*Tactical Security — fast field tools for authorized testing, triage, and analysis.*

## Why

TOKENROTATE exists for one job — plan + track secret rotation across providers from an inventory — and does it without a SaaS bill or heavyweight setup.
Single-purpose, scriptable, CI-friendly, self-hostable, and callable by AI agents over MCP.

## Install

```bash
pip install cognis-tokenrotate
# or from this repo:
pip install -e ".[dev]"
```

## Quick start

```bash
tokenrotate --version
tokenrotate scan .                      # scan the current project
tokenrotate scan . --format json
tokenrotate scan . --fail-on high       # non-zero exit for CI gates
tokenrotate mcp                         # expose as an MCP server (Cognis.Studio / Claude Desktop / Cursor)
```
